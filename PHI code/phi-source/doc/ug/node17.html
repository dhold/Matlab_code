Compiling Phi

Download the PHI source code from www.ks.uiuc.edu/Research/phi. Then:

  1. Unzip and untar.

  2. Edit make.config in the source root directory to
    - provide paths to CBLAS libraries and headers
    - specify computer architecture (linux, mac, solaris)
    - choose compiler (gcc, icc, sun CC)

  3. ./make install to install the phi binary to ./bin/
