Environment parameters

  • Temperature
    Acceptable Values: positive real value
    Default Value: 300 Kelvin
    Description: Specifies the temperature of the environment. Units are in Kelvin.
    Example line(s):
    Temperature=300
  • MatsubaraTerms
    Acceptable Values: non-negative integers
    Default Value: 0
    Description: Specifies the number of terms to include for the Matsubara expansion.
    Example line(s):
    MatsubaraTerms=4
