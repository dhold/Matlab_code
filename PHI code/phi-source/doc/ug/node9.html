System parameters

The system parameters specify the quantum states and matrix elements of the Hamiltonian .

  • NumStates
    Acceptable Values: positive integers
    Default Value: 0
    Description: Specifies the number of quantum states [required]
    Example line(s):
    NumStates=2
  • Hamiltonian
    Acceptable Values: real values
    Default Value: None
    Description: Specifies the system Hamiltonian. Note that the renormalization term must be included here
    Example line(s):
    Hamiltonian:
    0 ,100.0
    100.0,20.00
  • InitialDensityMatrix
    Acceptable Values: real or complex values
    Default Value: None
    Description: Specifies the initial values of the density matrix. The initial density matrix must be positive definite.
    Example line(s):
    InitialDensityMatrix:
    (0.5,0.0),(0.2,0.1)
    (0.2,-0.1),(0.5,0.0)
  • HamiltonianEigenvalues
    Acceptable Values: real values
    Default Value: None
    Description: Eigenvalues of system Hamiltonian [required for TimeLocal=1].
    Example line(s):
    HamiltonianEigenvalues:
    -90.49875621,110.49875621
  • HamiltonianEigenvectors
    Acceptable Values: real values
    Default Value: None
    Description: Orthonormal eigenvectors of system Hamiltonian [required for TimeLocal=1].
    Example line(s):
    HamiltonianEigenvectors:
    -0.74145253,0.67100532
    0.67100532,0.74145253
