System-environment coupling parameters

The system-environment coupling is based on the Drude spectral density

(24)

Diagonal (independent) coupling, i.e. and , is assumed unless DiagonalCouplingIndices or CorrelatedCouplingTerms are specified.

  • NumCouplingTerms
    Acceptable Values: positive integers
    Default Value: 0
    Description: Specifies the number of system-bath coupling terms.
    Example line(s):
    NumCouplingTerms=2
  • lambda
    Acceptable Values: real values
    Default Value: None
    Description: Specifies the reorganization energy associated with each coupling term . Units are cm
    Example line(s):
    lambda:
    100.,50.0
  • gamma
    Acceptable Values: real values
    Default Value: None
    Description: Specifies the response frequency associated with each coupling term . Units are ps
    Example line(s):
    gamma:
    10,10
  • DiagonalCouplingIndices
    Acceptable Values: non-negative integers
    Default Value: None
    Description: Specifies the state to which the term couples. Note: state numbering starts from 0.
    Example line(s):
    #For NumStates=2 and NumCouplingTerms=4
    DiagonalCouplingIndices:
    0,0,1,1
  • CorrelatedCouplingTerms
    Acceptable Values: real values
    Default Value: None
    Description: Specifies each system-environment coupling .
    Example line(s):
    #For NumStates=2 and NumCouplingTerms=3
    CorrelatedCouplingTerms:
    1,0
    0,1
    0.7071,-0.7071
