Hierarchy parameters

  • HierarchyTruncation
    Acceptable Values: integer 1
    Default Value: 0
    Description: Specifies the level to truncate the hierarchy.
    Example line(s):
    HierarchyTruncation=5
  • TimeLocal use time local truncation?
    Acceptable Values: 0 or 1
    Default Value: 1
    Description: Specifies whether to use time local or time non-local truncation.
    Example line(s):
    TimeLocal=1
