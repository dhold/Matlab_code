PHI is a parallel, object-oriented code for integrating the hierarchy
equations of motion for dissipative quantum dynamics. 
PHI is distributed free of charge and includes source code.

All distributions of PHI include the following files:

 notes.txt
	This file includes directions for compiling and running PHI

 license.txt
	This file contains the license under which PHI is distributed
	and to which you must agree in order to use the program.

Full documentation for using PHI is available from the PHI web site:
	http://www.ks.uiuc.edu/Research/phi/
